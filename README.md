# Dev Calc 1

## Objetivo do Projeto

Este é um projeto de demonstração para o Spring Boot, que implementa uma calculadora simples.

## Ferramenta de Build

O projeto utiliza o [Apache Maven](https://maven.apache.org/) como ferramenta de build.

## Como Executar Localmente

Para executar a aplicação localmente, utilize o seguinte comando na raiz do projeto:

```bash
./mvnw spring-boot:run
```

A aplicação estará disponível em `http://localhost:8080`.
